# CMS BLOG FRONTEND

### RUN THE PROJECT LOCALY

Clone the project

```bash
  git clone https://github.com/Warisaremou/cms-blog-frontend.git
```

Go to the project directory

```bash
  cd cms-blog-frontend
```

Install dependencies

```bash
  npm install
```

Add .env file

```bash
  cp .env.example .env
```

Setup hooks for pre-commit

```bash
  npm run prepare
```

Run development server

```bash
  npm run dev
```
